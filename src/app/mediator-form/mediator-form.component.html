Simple Instructions
Command Description
begin Starts the connection with the robot (Must start from here to test the demo).
done Closes the connection to the robot.
close Closes the connection to the server, server stops.
forward Moves forward for a certain time.
backward Moves backwards for a certain time.
right Rotates the robot to the right.
left Rotates the robot to the left.
Complex Instructions
Command Description
Parameter Number that the robot will execute a certain instrucction.
Type Type of the parameter. If time, the parameter will be in seconds.
done Closes the connection to the robot.
close Closes the connection to the server, server stops.
forward The robot will move forward (if parameter type is time, it will move for the indicated time).
backward The robot will move backwards (if parameter type is time, it will move for the indicated time).
right The robot will rotate to the right (if parameter type is time, it will rotate for the indicated time,
if it's angle, the robot will rotate by the indicated angle).
left The robot will rotate to the left (if parameter type is time, it will rotate for the indicated time,
if it's angle, the robot will rotate by the indicated angle).
